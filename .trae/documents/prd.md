## 1. Product Overview
北京祥云圣火航空科技有限公司官方网站首页，展示公司业务能力和服务项目，提升品牌形象和业务推广。
- 展示公司核心业务和服务内容
- 提供联系方式和公司信息
- 面向航空航天领域客户和合作伙伴

## 2. Core Features

### 2.1 Feature Module
1. **首页**: 顶部导航、Hero轮播区、服务展示、底部信息

### 2.2 Page Details
| Page Name | Module Name | Feature description |
|-----------|-------------|---------------------|
| 首页 | 顶部导航 | 公司logo、名称、导航菜单、移动端菜单按钮 |
| 首页 | Hero轮播区 | 自动轮播展示公司业务能力，带标题和背景图 |
| 首页 | 服务展示区 | 左侧导航菜单 + 右侧服务项目卡片展示 |
| 首页 | 联系浮窗 | 右侧固定联系按钮（邮箱、电话、在线咨询） |
| 首页 | 底部信息 | 公司链接、联系方式、版权信息 |

## 3. Core Process
用户访问首页 → 浏览导航菜单 → 查看Hero区业务能力 → 浏览服务项目 → 通过联系按钮获取更多信息

```mermaid
flowchart LR
    A[用户访问首页] --> B[浏览导航菜单]
    B --> C[查看Hero轮播区]
    C --> D[浏览服务展示区]
    D --> E[点击联系按钮]
    E --> F[获取联系方式]
```

## 4. User Interface Design
### 4.1 Design Style
- 主色调：深蓝色(#1a365d) + 红色点缀(#dc2626)
- 按钮风格：圆角矩形，hover效果
- 字体：中文使用微软雅黑/思源黑体，英文使用Arial
- 布局风格：顶部导航 + Hero区 + 左右布局服务展示 + 底部信息
- 图标风格：简洁现代

### 4.2 Page Design Overview
| Page Name | Module Name | UI Elements |
|-----------|-------------|-------------|
| 首页 | 顶部导航 | 白色背景，logo居左，导航菜单居中，移动端汉堡菜单居右 |
| 首页 | Hero轮播区 | 全屏大图背景，半透明遮罩，白色大标题文字，自动轮播 |
| 首页 | 服务展示区 | 左侧红色高亮导航菜单，右侧卡片式服务展示，带图片和标题 |
| 首页 | 联系浮窗 | 右侧固定三个圆形按钮（邮箱、电话、咨询），hover动画 |
| 首页 | 底部信息 | 深灰色背景，左侧公司链接列表，右侧logo和联系方式 |

### 4.3 Responsiveness
- 桌面端(>1200px)：完整左右布局
- 平板端(768-1200px)：服务卡片改为两列布局
- 移动端(<768px)：左侧菜单折叠为汉堡菜单，服务卡片单列展示

### 4.4 3D Scene Guidance
- 不适用
